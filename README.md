# 🕒 Relógio Digital Interativo

Relógio digital moderno que exibe o horário em tempo real com interface estilizada e responsiva.
Projeto desenvolvido para praticar manipulação de DOM, estilização com CSS e lógica com JavaScript.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Funcionalidades

✔ Exibição da hora em tempo real
✔ Atualização automática a cada segundo
✔ Interface moderna e minimalista
✔ Layout responsivo para desktop e mobile

## 🌐 Acesse o Projeto Online

🔗 https://lkaua370.github.io/RelogioDigitalAtt/

## 💻 Como Executar Localmente

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador

## 📚 Aprendizados

Este projeto ajudou a reforçar conceitos importantes como:

* Manipulação de elementos com JavaScript
* Funções de tempo (`setInterval`)
* Organização de estrutura HTML
* Estilização moderna com CSS
